## Environment
* Ubuntu
* macOS

## Installation
### Ubuntu
```bash
git clone https://github.com/ToshikiNakamura0412/vim.git
./vim/install.sh
```

### macOS
```bash
git clone https://github.com/ToshikiNakamura0412/vim.git
./vim/install.sh mac
```
